#### 软件架构
参照ABP框架采用Autofac实现模块化依赖注入，包含重写Session、接口报错数据库日志、UOW+Repository模式实现、Redis的ServiceStack封装等功能，
同时也结合AngularJS实现了一套角色权限管理功能
